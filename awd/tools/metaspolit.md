# 1.show auxiliary
显示辅助模块

- /scanner 扫描软件
  - /smb smb服务相关
    - /smb_version 扫描smb版本（根据banner）
  - /ssh ssh服务相关
    - /ssh_version 扫描ssh版本（根据banner）
  - /ftp ftp服务相关
    - /anonymous 扫描ftp匿名服务
  - /snmb snmb服务
    - /snmb_enum 扫描开启snmb的主机


# 2.show explits

# 3.show options

# 4.show paylaods

# 5.show targets

列出收到漏洞影响目标系统的类型。
